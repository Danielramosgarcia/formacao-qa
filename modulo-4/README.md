# Módulo 4 – Testes Manuais em Sites Públicos

Este módulo tem como objetivo praticar testes manuais em sites reais,
focando em interface, comportamento e fluxo do usuário.

## Escopo dos testes
- Testes funcionais
- Testes de interface (UI)
- Testes de formulário
- Testes de login

## Sites testados
- Mercado Livre
- Amazon Brasil
- LinkedIn
- Magazine Luiza

## Entregáveis
- Casos de teste positivos e negativos
- Cenários de teste
- Evidências (prints)
- Bug reports (quando aplicável)
